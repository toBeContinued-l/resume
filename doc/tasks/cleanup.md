- [x] 实现生成完成后的临时任务目录删除。
- [x] 实现 AI 处理失败后的临时任务目录删除。
- [x] 实现解析重试耗尽后的临时任务目录删除。
- [x] 实现上传接口失败时的部分临时文件清理。
- [x] 实现清理后任务状态更新为 `cleaned` 或按流程保留 `completed`。
- [x] 实现解析重试耗尽后失败简历记录对用户不可见。
- [x] 实现清理操作幂等，重复执行不影响其他任务。
- [x] 实现清理失败日志记录，保留后续重试所需信息。
- [x] 确保清理服务只能删除受控任务目录。
- [x] 在 Worker 成功、失败和重试耗尽分支中接入清理服务。
- [x] 编写成功任务清理测试。
- [x] 编写失败任务清理测试。
- [x] 编写重试耗尽不保留用户可见失败记录测试。
- [x] 编写清理幂等和路径安全测试。

## 完成标准

- [x] 原始上传文件不会在生成完成或失败后长期保留。
- [x] 清理失败可被发现并再次执行。
- [x] 清理逻辑不会误删其他用户或其他任务目录。
