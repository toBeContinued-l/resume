# 账号认证模块任务

## 最小可执行任务

- [x] 定义用户、邮箱验证令牌、密码重置令牌、会话的数据模型与迁移脚本。
- [x] 实现邮箱格式、密码强度、邮箱唯一性校验。
- [x] 实现密码哈希与校验服务，当前使用 Node `scrypt`，密码和令牌均不明文持久化；后续生产可按详细设计替换为 `argon2id` 或 `bcrypt`。
- [x] 实现注册 API：创建 `pending_verification` 用户并生成邮箱验证令牌和 6 位邮箱验证码。
- [x] 实现邮件 Provider 接口，并接入邮箱验证码发送流程；注册页同页回填验证码，不依赖邮件跳转。
- [x] 实现邮箱验证 API：校验验证码哈希、过期时间和重复使用状态，并保留令牌兼容入口。
- [x] 实现登录 API：校验账号状态、密码哈希并创建登录会话。
- [x] 实现退出登录 API：撤销当前会话并清理 Cookie。
- [x] 实现当前用户 API：返回登录用户基础信息。
- [x] 实现密码找回 API：生成一次性重置令牌并发送邮件。
- [x] 实现重置密码 API：校验重置令牌并更新密码哈希。
- [x] 实现认证中间件或服务方法，用于保护后台 API。
- [x] 实现简历所有权校验服务，供编辑、历史记录、上传限制和删除使用。
- [x] 设置会话 Cookie 的 `HttpOnly`、`Secure`、`SameSite=Lax` 属性。
- [x] 在注册页面展示用户协议和隐私政策入口。
- [x] 编写账号认证单元测试：注册、验证、登录、退出、找回密码、令牌过期。
- [x] 编写权限测试：未登录返回 401，非所有者访问简历返回 403。

## 完成标准

- [x] 邮箱注册、验证、登录、退出和密码找回主流程可运行。
- [x] 所有后台接口都能复用登录态和所有权校验。
- [x] 密码、会话令牌、邮箱验证令牌和密码重置令牌均不明文持久化。
